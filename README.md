# Thermal-Fluid-Modeling-and-its-surrogate-model-of-a-Piston-Engine-for-Large-UAVs-2026
The source model based on GT-Power
Authors: Wennian Yu
Journal: Applied Science
Corresponding Author: Wennian Yu (wennian.yu@cqu.edu.cn)
# 仓库内容
1. 变高度无涡轮增压.gtm: The souce model of a Piston Engine without turbocharger
2. 变高度无涡轮增压.gdx: The output data after runing the .gtm model
3. 4cyl.gtm: The surrogate model
4. 4cyl.gtx: The output data after runign the .gtm model
5. 代理模型输入参数表.xlsx: The input parameters of the surrogate model
## 软件环境依赖
1. GT-SUITE 2023/2024（GT-Power核心求解器，需商业授权软件打开模型）
2. MATLAB R2022b及以上（运行自动化仿真脚本）
3. Python 3.9（绘图后处理）
